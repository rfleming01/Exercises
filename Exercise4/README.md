# Platforming Exercise
**Version: 0.1**
* Player character moves around with arrow keys and space bar.
* platforms can be stood on, including the moving platform, which moves up and down.
* The screen follows the player around, but stays within the map boundries.
* Coins can be collected, and the amount is displad on the hud.
